- [W124 紅外線遙控器重置](https://www.mobile01.com/topicdetail.php?f=303&t=3239386)
  1. 鑰匙開副駕門進到主駕位置。
  2. 關好所有門以及開好主駕車窗，保持key off。
  3. 開明火 => Key Off => 拔鑰匙後對主駕門外的紅外線按住鑰匙不放。
  4. 若成功會看到門上紅外線裝置閃幾下紅燈，當停止閃動後，應該就完成鑰匙重置動作，試看看應該可以用遙控開關車門了。
